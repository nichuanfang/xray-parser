## 使用手册


### 启动`xray`

```bash
sudo chmod +x ./restart.command
./restart.command
```
### 关闭代理

```bash
sudo chmod +x ./disable_proxy.command
./disable_proxy.command
```
### 启动代理

```bash
sudo chmod +x ./enable_proxy.command
./enable_proxy.command
```
### 更新配置

```bash
# 更新脚本中的用户名和密码
sudo chmod +x ./update_config.command
./update_config.command
```
